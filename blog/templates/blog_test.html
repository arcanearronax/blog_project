Prove: If \(p\) is a prime and \(n\) a positive integer, then \(p^{n}={\displaystyle \sum_{k=0}^{n}\phi(p^{k})}\)

To show this, we use induction on \(n\). We begin by showing the base case \(n=0\). $$p^{0}=1={\displaystyle \sum_{k=0}^{0}}\phi(p^{k})=\phi(p^{0})=\phi(1)=1$$ so the base case is proven. Now we assume the case for \(n\) to be true and show the case of \(n+1\) is true. $$p^{n+1}={\displaystyle \sum_{k=0}^{n+1}}\phi(p^{k})=1+{\displaystyle \sum_{k=1}^{n+1}}p^{k-1}(p-1)=1+(p-1)+p(p-1)+...+p^{n-1}(p-1)+p^{n}(p-1)=(1-1)+(p-p)+(p^{2}-p^{2})+...+(p^{n-1}-p^{n-1})+p^{n}=p^{n}$$ So we have shown that if \(p\) is prime and \(n\in\mathbb{N}\) then $$p^{n}={\displaystyle \sum_{k=0}^{n}\phi(p^{k})}.$$
